# Nexus Landing Page

A responsive one-page landing page built with HTML, CSS, and JavaScript.

## Overview

This project contains a modern marketing-style landing page for a fictional product called **Nexus**. It includes:

- hero section with call-to-actions
- animated loader and reveal effects
- sticky navigation with active section highlighting
- responsive mobile hamburger menu
- contact form validation
- feature showcase cards and stats

## Files

- `index.html` - main page markup
- `style.css` - page styling and responsive layout
- `script.js` - interactive behavior for loader, navigation, animations, and contact form

## Usage

1. Open `index.html` in a web browser.
2. The page is fully static and does not require a server for local preview.
3. For development, edit the HTML, CSS, or JavaScript files and refresh the browser.

## Notes

- The contact form uses client-side validation and a simulated submission flow.
- The project uses Google Fonts for typography.
